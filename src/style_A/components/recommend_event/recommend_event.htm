外貿協會於2020年欣逢成立半世紀，7月7日於台北國際會議中心辦理成立50週年茶會，茶會見證了貿協50年來輝煌的歷程，也宣示新時代的使命。

未來貿協將不負國家交付的使命，以及各界的期待，將永遠走在最前線，與台灣企業攜手打拼，繼續航向未來五十年，共創台灣經貿新榮景。
